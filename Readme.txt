This code demonstrate the Secure Multi-party Computation.
It uses the Shamir's secret sharing in decentralize environment.
For input you should use the input file.
The format of the file is in csv format.
there will be only two lines in the input file.
The first line will be of public values of all the participating parties.
The second line will be the secret value of them.

e.g. 
2,3,4,5,6,8
12,24,102,33,55,64

If line number 1 and line number 2 does not have same number of values the program will exit.
